
# DummyJSON Users Table (React)

Приложение реализующее таблицу пользователей с сортировкой, фильтрацией, постраничной навигацией и модальным окном с подробностями. Используется API: https://dummyjson.com/docs/users

## Возможности
- Таблица с колонками: Фамилия, Имя, Отчество, Возраст, Пол, Телефон, Email, Страна, Город
- Сортировка по колонкам (через query params `sortBy` + `order` на API) — циклично: без сортировки → по возрастанию → по убыванию
- Поиск / фильтрация (локальный фильтр по ФИО, email, телефону, городу)
- Пагинация
- Модальное окно с подробной информацией о пользователе
- Обработка ошибок запросов и индикатор загрузки
- Изменяемая ширина колонок
